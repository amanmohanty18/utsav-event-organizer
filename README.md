# 🪔 Utsav – Event Organizer & Ticket Booking Platform

🚀 A modern responsive event management web application for discovering, hosting, and booking cultural, music, corporate, sports, and festival events across India.

🔗 Live Demo: [View Website](https://amanmohanty18.github.io/utsav-event-organizer)

---

## 📌 About The Project

Utsav is a modern event organizer and ticket booking platform designed to simplify event discovery, management, and RSVP booking. The platform provides an interactive and user-friendly experience with dynamic event rendering, category filtering, ticket booking functionality, authentication UI, and responsive design.

This project demonstrates frontend development skills using core web technologies while implementing real-world functionalities such as event management, local storage integration, search filtering, and responsive UI/UX.

---

## ✨ Features

- 🎉 Browse events across multiple categories
- 🔍 Search events by city and category
- 🎟️ RSVP & ticket booking system
- 🔐 Login & signup authentication UI
- 📱 Fully responsive design
- ⚡ Interactive modern UI/UX
- 💾 Local storage integration
- 📊 Dynamic event rendering
- 🎨 Gradient-based event cards
- 📬 Contact & organizer interface
